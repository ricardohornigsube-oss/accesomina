Publicar en cloud / URL pública
Estado actual: esta versión funciona como app HTML local con datos separados por cliente en este computador. Para publicarla en internet de forma segura, Domian debe activar backend PHP o Node, base de datos, login con contraseñas, almacenamiento privado de archivos, SSL y backups.
Opción recomendada
Migrar a Laravel/PHP + MySQL/PostgreSQL. Cada cliente queda con un tenant_id propio, usuarios por empresa, permisos y archivos separados.
Respaldo portable
Usa el botón Respaldo para descargar toda la data del cliente actual en JSON y cargarla luego en un servidor.
