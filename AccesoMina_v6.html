Domian · Tecnología para contratistas mineras

Acreditación minera, contratos y alertas en un solo sitio privado.

AccesoMina ordena trabajadores, proyectos, hoteles, mantenciones, exámenes, cursos, contratos y auditoría documental para empresas contratistas que necesitan ingresar personal a faenas mineras sin perder control de vencimientos ni respaldos.

Usuarios por empresaCada empresa administra su equipo con permisos de administrador, edición o solo lectura.
Datos totalmente separadosEmpresa A y Empresa B operan en espacios independientes; su información nunca se mezcla.
Preparado para trabajo simultáneoLa estructura multiempresa está lista para backend, base de datos, sesiones concurrentes y URL segura.

Publicar en cloud / URL pública
Estado actual: esta versión demuestra la separación por empresa, usuarios y permisos en almacenamiento local. Para que 10, 20 o más personas trabajen realmente al mismo tiempo desde distintos equipos, se debe activar backend PHP o Node, base de datos multiempresa, contraseñas seguras, archivos privados, SSL y backups.
Arquitectura recomendada
Laravel/PHP o Node + MySQL/PostgreSQL. Cada empresa conserva un tenant_id propio, usuarios, roles, sesiones y archivos completamente separados.
Respaldo portable
Usa Respaldo para descargar la data de la empresa actual y migrarla posteriormente al servidor privado.
